Setting up /setrank


What this does

The /setrank command lets you promote or demote guild members both in Discord (by changing their role) and in Wynncraft in-game at the same time. To make it work you need to link your Discord roles to the rank names in the config file.


Step 1 - Enable Developer Mode in Discord

You need Developer Mode on to copy role IDs.

1. Open Discord
2. Click the settings cog at the bottom left
3. Click Advanced in the left sidebar
4. Turn on Developer Mode


Step 2 - Create ranks as Discord roles (if you haven't already)

Go to your Discord server, click the server name at the top, then Server Settings, then Roles. Create roles for each Wynncraft rank you want to track. For example: Recruit, Member, Captain, Chief. You can name them whatever you want but keep track of which is which.


Step 3 - Get the role IDs

For each rank role you created, right click on the role name and click Copy Role ID. Write them down like this:

Recruit role ID: (paste here)
Member role ID: (paste here)
Captain role ID: (paste here)
Chief role ID: (paste here)


Step 4 - Update config.json

Open C:\Users\YourUsername\WynnTrackerServer\config.json in Notepad and find the ranks section at the bottom. Replace each placeholder with the ID you copied:

  "ranks": {
    "recruit": {
      "discord-role-id": "PASTE_RECRUIT_ROLE_ID_HERE",
      "identifier": "Recruit",
      "ingame-rank": 1
    },
    "member": {
      "discord-role-id": "PASTE_MEMBER_ROLE_ID_HERE",
      "identifier": "Member",
      "ingame-rank": 2
    },
    "captain": {
      "discord-role-id": "PASTE_CAPTAIN_ROLE_ID_HERE",
      "identifier": "Captain",
      "ingame-rank": 4
    },
    "chief": {
      "discord-role-id": "PASTE_CHIEF_ROLE_ID_HERE",
      "identifier": "Chief",
      "ingame-rank": 5
    }
  }

Save the file.


Step 5 - Restart the server

After saving config.json, restart the server for the changes to take effect:

cd C:\Users\YourUsername\WynnTrackerServer
npm run start


Step 6 - Using /setrank

In Discord, type /setrank and select a member and the rank you want to give them. The bot will update their Discord role immediately and also attempt to promote them in-game if someone with a high enough rank is online and connected to the mod.

If nobody eligible is online when you run the command, it will queue the promotion and retry automatically every minute until it goes through.


Notes

The ingame-rank numbers correspond to Wynncraft guild ranks:
1 = Recruit
2 = Member
3 = Squire (not used by default)
4 = Captain
5 = Chief
6 = Owner (cannot be set through the bot by design)

The identifier field is just the display name shown in Discord messages. You can change it to whatever you want.
