WynnTrackerServer Setup Guide (Pre-configured version)



What you need before starting

Node.js installed, and MySQL installed.


Step 1 - Install Node.js

Go to nodejs.org and download the LTS version. Run the installer and just click Next through everything. When it is done restart your computer or at least close and reopen any command prompt windows you have open.

To check it worked, open Command Prompt (search for cmd in the start menu) and type:
node --version

You should see a version number like v22.0.0.


Step 2 - Install MySQL

Go to dev.mysql.com/downloads/mysql/ and download the MSI Installer for Windows. When you run it, choose Server only as the setup type. Keep clicking through until you get to the Accounts and Roles page, this is where you set a root password (remember cus you'll need it for later). Finish the installation.

MySQL will now run automatically in the background every time your PC starts.


Step 3 - Create the database

Open Command Prompt and type this to open MySQL:

"C:\Program Files\MySQL\MySQL Server 8.4\bin\mysql.exe" -u root -p

It will ask for your password. Type the one you set during installation.

Once you see the mysql prompt, type:
CREATE DATABASE wynntracker;

You should see Query OK. Then type exit to close it.


Step 4 - Install the server dependencies

Put the WynnTrackerServer folder somewhere on your PC, for example:
C:\Users\YourUsername\WynnTrackerServer

Open Command Prompt, navigate to the folder and install dependencies:

cd C:\Users\YourUsername\WynnTrackerServer
npm install

If you get an error saying running scripts is disabled, run this first:
Set-ExecutionPolicy -Scope CurrentUser -ExecutionPolicy RemoteSigned

Then press Y and Enter, and try npm install again. Or just use regular Command Prompt instead of PowerShell.


Step 5 - Create a Discord bot

Go to discord.com/developers/applications and log in.

Click New Application in the top right and give it a name like WynnTracker.

On the General Information page, find the Application ID and copy it somewhere. This is your client ID.

Click on Bot in the left sidebar. Click Reset Token, confirm it, and copy the token. Save this somewhere safe because you can only see it once.

Scroll down and turn on both Server Members Intent and Message Content Intent then hit Save Changes.

To add the bot to your server, go to OAuth2 then URL Generator. Check the bot and applications.commands boxes under Scopes, then check Administrator under Bot Permissions. Copy the URL at the bottom and open it in your browser to invite the bot to your server.


Step 6 - Set up the config file

Create a new file called config.json inside the WynnTrackerServer folder and paste this in, FILL IN YOUR OWN VALUES:

{
  "token": "YOUR_DISCORD_BOT_TOKEN",
  "clientId": "YOUR_DISCORD_CLIENT_ID",
  "guild-tag": "YOUR_GUILD_PREFIX",
  "minimum-rank": 2,
  "wynncraft-token": "WYNNCRAFT_API_TOKEN",
  "host-port": 8080,
  "account-linking": {
    "required-role-id": "REQUIRED_ROLE_ID_TO_USE_LINK_COMMAND",
    "linked-role-id": "ROLE_ID_GIVEN_TO_LINKED_PLAYERS",
    "verification-expiry-minutes": 15
  },
  "sql": {
    "host": "localhost",
    "user": "root",
    "password": "YOUR_MYSQL_PASSWORD",
    "database": "wynntracker"
  },
  "chat-bridge": {
    "enabled": true,
    "webhook-url": "https://discord.com/api/webhooks/YOUR_WEBHOOK_ID/YOUR_WEBHOOK_TOKEN",
    "channel-id": "YOUR_CHANNEL_ID"
  },
  "alert-command": {
    "required-role-id": "ROLE_ID_REQUIRED_FOR_ALERT_COMMAND"
  },
  "ranks": {
    "recruit": {
      "discord-role-id": "RECRUIT_ROLE_ID",
      "identifier": "Recruit",
      "ingame-rank": 1
    },
    "member": {
      "discord-role-id": "MEMBER_ROLE_ID",
      "identifier": "Member",
      "ingame-rank": 2
    },
    "captain": {
      "discord-role-id": "CAPTAIN_ROLE_ID",
      "identifier": "Captain",
      "ingame-rank": 4
    },
    "chief": {
      "discord-role-id": "CHIEF_ROLE_ID",
      "identifier": "Chief",
      "ingame-rank": 5
    }
  }
}

The fields you must fill in are token, clientId, guild-tag, and the sql password. Leave wynncraft-token as it is, it does not need a real value.


Step 7 - Run the server

Open Command Prompt and run:

cd C:\Users\YourUsername\WynnTrackerServer
npm run start

When you see Discord bot ready it means everything is working. Keep this window open while you play. To stop the server just close the window or press Ctrl and C.

You need to run this every time you want to track raids.


Step 8 - Install the mod

Players that want their raids tracked will need to have the mod (1 player at least per raid maybe if someone else with the mod is on and nobody in the raid has the mod it'll still work but I'm not sure)

Put the jar file in your Minecraft mods folder and make sure you are using the Fabric loader.

Once in game, open Mod Menu, find WynnTracker, open its settings, and enter the server URL. If you are playing on the same PC that runs the server use http://localhost:8080. If you are connecting to someone else's server ask them for their public IP and use http://THEIR_IP:8080.


Discord bot commands

1. /raids followed by a player name - shows how many guild raids that player has completed
2. /leaderboard - shows the guild raid leaderboard
3. /gxp - shows guild XP rankings
4. /aspects followed by a player name - shows how many aspects that player is owed
5. /owed - shows all players who are currently owed aspects
6. /giveaspect followed by a player name and amount - records that you gave a player aspects, reduces their owed count (admin only)
7. /setaspects followed by a player name and amount - manually sets how many aspects a player is owed, useful for corrections (admin only)
8. /setrank followed by a member and rank - sets a member rank in Discord and in game (admin or ranked members only)


Common problems

If npm is not recognized, restart Command Prompt after installing Node.js.

If you get connection refused errors when starting the server, MySQL might not be running. Open Task Manager, go to the Services tab, find MySQL84 and right click it to start it.

If the Discord bot says Unauthorized, double check your token in config.json. If it still does not work go to the Discord Developer Portal and reset the token, then update config.json with the new one.

If you get Failed to get token from server in game, make sure you are actually logged into a Wynncraft server and not just sitting on the Minecraft main menu. Also double check the URL in the mod settings is exactly http://localhost:8080 with no extra spaces or slashes.

If you get Player is not in the guild errors, double check your guild-tag in config.json. The capitalization needs to match exactly what Wynncraft shows on their website.

If you get Failed to report raid after a guild raid, try doing another one. It sometimes happens on the first attempt after a fresh install.
